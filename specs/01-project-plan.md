# SprintQuest Project Plan

## Project Summary

SprintQuest is a gamified project development canvas board for managing software projects through projects, sprints, tasks, checklist items, XP, achievements, and progress tracking.

## Main Goal

Build a full-stack web application that satisfies the MSA Phase 2 Software Stream requirements while demonstrating a clear gamification theme.

## Milestone Plan

- M0 - Planning & Repository Setup
- M1 - Backend Solution Setup
- M2 - Core Domain Models
- M3 - EF Core Database
- M4 - CRUD API & Scalar Docs
- M5 - Frontend Setup
- M6 - Sprint Board UI
- M7 - Gamification System
- M8 - Zustand State Management
- M9 - Security Measures
- M10 - SignalR Live Board
- M11 - Testing
- M12 - Deployment
- M13 - README, Specs & Video
- M14 - Final Release
